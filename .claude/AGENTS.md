# Unze Cockpit — Custom Agents

Run any agent with: /agents <name>

| Agent | Command | What it does |
|---|---|---|
| Blueprint Keeper | /agents blueprint-keeper | Updates BLUEPRINT.md and CHANGELOG.md |
| UI Designer | /agents ui-designer | Design improvements and styling |
| Session Summarizer | /agents session-summarizer | Summarises current session |
| Build Guardian | /agents build-guardian | Watches for build/TS errors |
| Code Auditor | /agents code-auditor | Reviews code quality |
| Database Architect | /agents database-architect | Postgres/Supabase schema work |
| Data Integration | /agents data-integration | Handles data/API integrations |
| Ideas Scout | /agents ideas-scout | Explores feature ideas |
