Reflow & new components

Desktop → Mobile
01 · NEW
Form card (long-form)
Card with a bordered head (icon mark + title + optional chip), then a grid of labeled fields, then a footer with an informative hint on the left and cancel/primary buttons on the right.
02 · NEW
Field group patterns
3-col row (task desc [2fr] + assignee + priority), 3-col row for scheduling, 2-col row for optional tags. On mobile, all fields stack single-column except priority + due-after which pair up.
03 · NEW
Example / starter cards
3×2 grid of pre-filled templates on desktop (production report, vendor payment run, etc.). Each card is a "start from here" affordance — clicking pre-fills the form.
04
Rich empty state
Instead of your current one-line "No recurring tasks set up yet", the empty state now has icon + headline + body + a whole starter grid. Turns a dead-end into onboarding.
05
Informative footer hint
"Next occurrence: Mon 06 Jul 2026 · Assignee will get a task 3 days before it's due." Live preview of what the template will actually do — reduces surprises.
06
Mobile form density
Desktop uses a 3-col form. Mobile keeps the field grouping intact but stacks them, with tight pairing (priority + due days on one row) preserved for related concepts.
