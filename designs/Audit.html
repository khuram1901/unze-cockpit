Reflow & new components

Desktop → Mobile
01 · NEW
Horizontal bar chart
Right-aligned label · thin 14px track · colored fill · value. Rows use color to signal stage grouping: accent (planning), warning (in progress), purple (review), success (final).
02 · NEW
Stacked workload bar
Per-auditor stacked bar with segments for overdue / in progress / complete. Total count on the right. Legend row at the bottom.
03 · NEW
Audit record row
4-column: title + meta (with type badge + auditor + target), progress bar with %, status pill, action icon. Uses the same grouped-list header pattern from Admin.
04 · NEW
Type badge (audit)
Compact 10px pill for audit type. Financial=accent, Operational=success-tinted, Compliance=purple. Purple is the only new hue introduced beyond the base palette — reused wherever "governance / legal / compliance" is signaled.
05
3-chart row → stacked
Stage pipeline + Type donut + Workload becomes a single donut card on mobile (the most compact summary), with the horizontal bar and workload available on a "Detail →" screen.
06
Record row → vertical card
Desktop 4-col grid becomes a stack: title + type badge, meta line, progress bar. Status shows via progress bar color and text color of "N d late" only.
