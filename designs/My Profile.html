Reflow & new components

Desktop → Mobile
01 · NEW
In-page sub-nav
Sticky left card of sub-sections (Account, Security, Notifications, Preferences, Danger). Active item mirrors sidebar styling — dark pill on white card. Reused wherever a settings-style page needs internal chapters.
02 · NEW
Settings card
Standard card + a head row (icon mark + title + optional status badge) with a hairline divider before the body. Every section on the page uses this anatomy — email, 2FA, password, notifications, push, danger.
03 · NEW
Toggle switch
40×22 pill with a 18px thumb. Off state is ink-300, on state is ink-900. Used wherever a boolean lives.
04 · NEW
Pref card (dual-control)
Checkbox + title/desc + toggle. The checkbox represents "is included in notification bundle"; the toggle represents "is the delivery channel on". Both are optional — most controls use just the toggle.
05 · NEW
Danger card
Card with a warm-red tint (#FDF3F1) and a red mark. Contains destructive actions. Very rare — only for sign-out-all, delete-account, and similar.
06
Two-col → single
Desktop 260px nav + fluid content becomes stacked. Nav collapses into a horizontal-scroll pill row (same pattern as page 06 stage filter).
