Reflow & new components

Desktop → Mobile
01 · NEW
Week grid
50px time column + 7 day columns, 44px per hour row. Empty cells get a subtle hover tint. Reusable for any time-based scheduling view (shifts, machine slots, etc.).
02 · NEW
Meeting block
Positioned absolutely inside a day cell. 3px left border indicates source: green = internal approved, blue = Google Cal, amber = pending. Title + meta line inside.
03 · NEW
Connected banner
Slim success-tinted banner confirming Google connection with event count. Reusable for any integration that's "healthy and connected".
04 · NEW
7-day strip (mobile)
One tile per weekday, active day inverts to dark, days with events get a small dot indicator below the number. Compact enough to fit 390px + touch-friendly.
05
Week grid → agenda list
Grid doesn't work on a phone. Mobile shows the 7-day strip up top, then a per-day agenda list below: each meeting is a card with time on the left, title/meta in the middle, chev on the right.
06
Empty pending state
Instead of your current one-liner "No pending meeting requests", the new empty state has a title + explanatory subtext, same treatment on both breakpoints.
