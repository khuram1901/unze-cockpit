How the layout reflows

Desktop → Mobile
This is the most data-dense page so far — but every element still uses primitives from the shared system. Cards marked NEW are additions to the design system.
01 · NEW
Entity switcher
Pill in the page header on desktop (logo + name + chev). On mobile it becomes a full-width bar directly under the top nav — the entity is *the* context, so it deserves the space. Tapping opens a sheet listing all entities.
02 · NEW
Editable metric card
Metric card + small pencil icon in the top-right (26×26 hit area, muted until hover). "Not set" numbers use ink-400 with a blue setup link underneath. Reused for any user-editable KPI going forward.
03 · NEW
Integration status card
List of external connections with logo, name + status pill, meta line, and per-connection action. Foot row shows ingestion counters. Reusable for every integration (Gmail today, banks / accounting software next).
04 · NEW
Dropzone + upload tabs
Dashed-border 14px-radius container with striped background hinting drop-behavior. Sits under a 2-tab strip (Upload / Manual entry). On mobile the dropzone shrinks to ~90px tall — tap opens the file picker directly.
05
4 KPIs → hero + 3 tiles
Desktop shows 4 equal-width metric cards, the last being a dark "Net position" hero. On mobile the hero jumps to the top as a full-width dark card, and the three "Not set" tiles become a compact 3-column row underneath — since they're setup tasks, not primary data.
06
Two-col forms → stacked
Ingestion + Add Position sit side-by-side on desktop; likewise Forecast + Budget. On mobile they stack single-column, with each card's inner controls kept intact (upload tabs, dropzone, budget toggles all present).
07
Charts stack
Line chart (cash balance) and bar chart (receipts vs payments) are 2-col on desktop. Stacked on mobile with narrower Y-axis labels and a smaller x-axis label density. Same colors, same legends.
08
Wide table → card list per month
7-column daily table (Date · Opening · Receipts · Payments · Closing · Post-dated · Net) becomes a mobile card list grouped by month header. Each row: date + inline ▲/▼ flow summary on the left, big Net number on the right.
09
Month divider row
Instead of one giant table, a subtle tinted divider row appears between months on desktop, and a small uppercase kicker between grouped cards on mobile — helps parse the timeline at a glance.
