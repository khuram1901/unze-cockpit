Unze OS · Design System v1

One system. Every page.

A calm, minimal design language for Unze Group's operations dashboard. Warm off-white canvas, restrained ink palette, big hero numbers, refined status colors. This document is the source of truth — every screen reuses these tokens and components.

Color

A limited, intentional palette. Ink is a deep navy, not black. Status colors are muted — they signal without shouting.

Grid & Reflow

Desktop uses a 12-column grid at 1440 max width, 32px gutters, 40px page padding. Mobile is a single column, 20px page padding, 16px card gap.
Desktop · 1440
12 cols · 32px gutter · 40px page padding · max-width 1440
Mobile · 390
4 cols · 16px gutter · 20px page padding · single-column flow
Reflow rules. Rows of 4 metric cards → 2×2 grid at tablet → single column at mobile. Wide charts + summary → chart on top, summary below. Tables → transformed to card-per-row on mobile (no horizontal scroll). Left nav → bottom tab bar on mobile.

Iconography

Lucide icon set at 1.5px stroke, 18px default size, always in Ink 500 unless active (Ink 900) or status-tinted. Never fill icons.
Sizes. 14 (inline), 18 (default), 22 (headers), 32 (empty states). Colors. Default Ink 500 · Active Ink 900 · Never full-color; use status only for meaningful signal.
