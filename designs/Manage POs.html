Reflow & new components

Desktop → Mobile
01 · NEW
Compact PO row
A denser variant of the Stock PO card — accent left rule, customer + #ID + status chip in the head, description + meta line in the body, inline action buttons on the right. Closed POs get an ink-grey rule and 60% opacity.
02 · NEW
Contractor row
Gradient circular mark + name + one-line meta (letters issued, collection rate). Reused for anywhere a company/party appears in a list (auditors, taxation counsel, vendors).
03 · NEW
Contractor perf row
Big card with name + big % headline, thin fill bar underneath (amber under 50%, green above), and a 4-column stat row. Foot line summarises breakdown (fully / partial / not started).
04
Contractor form
2×2 field grid with required/optional labels. "Name" is the only required field — footer hint spells this out so users don't over-think.
05
Reuses plant switcher
Same dark-active pill tabs from Stock. Mobile uses the same horizontally-scrolling chip strip.
06
Mobile: PO rows compress
Desktop 2-col row (body + inline actions) becomes single-column cards with two action buttons in a bottom row. Closed POs omit the action row entirely.
