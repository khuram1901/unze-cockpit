How the layout reflows

Desktop → Mobile
A form-heavy page. Introduces the stepper, grouped input card, PO allocation card, and progress rail — plus a running-totals bar and sticky save. Cards marked NEW are additions to the design system.
01 · NEW
Number stepper
44px tall control with − / value / + segments. Value is display-font, tabular-nums, ink-400 at zero. Focus ring is ink-900 border. Used anywhere a discrete integer is entered — quantities, hours, counts.
02 · NEW
Grouped input card
Standard card + a 3px accent-colored left rule. Rule color signals semantics: success (good stock, income), warning (broken stock, at-risk), danger (loss). Reusable for any form section that needs a status hint.
03 · NEW
Progress rail
Sticky left-column card showing steps of a multi-step form (Plant + date → Opening stock → Allocate → Save). Done steps get a black checkmark disc; the active step is outlined black; upcoming steps are muted. Great for any multi-step wizard.
04 · NEW
PO allocation card
Card with customer + PO number title, one-line description, 4-stepper row, notes textarea, and a foot showing "Allocated N of PO qty" with a thin progress bar. Reusable for any per-item breakdown (invoices, shipments, etc.).
05 · NEW
Sticky save bar
Floats above the last card, sticks to viewport bottom, shows a summary of unsaved changes on the left and primary/secondary actions on the right. On mobile it becomes a narrower floating bar above the tab bar.
06 · NEW
Info banner (accent)
Tinted accent-soft box with an accent-square icon, title, and body text. Distinct from the warning/alert (amber) — used for helpful explanations rather than urgent flags.
07
Two-col form → single column
Desktop uses a 380px sticky meta column + fluid form column. Mobile stacks them: meta becomes the top form fields, and the progress rail collapses into a 4-dot progress bar under the page title.
08
3-across stepper → stacked labeled rows
On desktop, the three ft sizes (31/36/45) sit side-by-side with labels above each stepper. On mobile they become labeled rows (60px label column + fluid stepper) — much easier to thumb one at a time.
09
Notes field pattern
On desktop every PO card has a Notes textarea. On mobile it moves behind a "+ Notes" tap-to-add link below the stepper rows to save vertical space, since 5 PO cards with textareas would balloon the scroll.
