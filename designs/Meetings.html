Reflow & new components

Desktop → Mobile
01 · NEW
Month group card
Dark ink-900 header with title + count meta + collapse chevron, then a stack of meeting items with hairline dividers. A stronger version of the group pattern from Admin.
02 · NEW
Meeting item (expanded / collapsed)
Head row is always visible (date · title + tags · progress bar · actions). Expanded state reveals summary + attendees + decisions/risks/opportunities + action items.
03 · NEW
Decisions / Risks / Opportunities columns
3 accent-left-rule cards: success (decisions), danger (risks), accent (opportunities). Each has a title + count + bulleted list. On mobile they stack — with the same rule/color code preserved.
04 · NEW
Attendee chip
Small rounded pill with an optional uppercase role prefix (e.g. "CEO Khuram Saleem"). Reusable for any people-tagged list — audit teams, task assignees, meeting requesters.
05 · NEW
Action item row
3-col: checkbox, title + due meta, status pill (same statuses as task rows in Admin). Sits inside its own card with a header that includes progress count.
06
Full doc → summary card
Desktop shows the entire minutes document expanded. Mobile compresses each meeting to a single card with date + title + tags + progress bar. Tapping opens the full expansion on a dedicated screen.
