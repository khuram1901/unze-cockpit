Reflow & new components

Desktop → Mobile
01 · NEW
Overdue group card
Standard task group with a full 3px danger-red left rule and warm-red tinted header. Reserved for groups where any task is overdue. Reusable pattern for any "requires action" group.
02 · NEW
Filter tab (with count)
Pill button with a mono count badge (Total / Urgent / High etc). Active pill inverts to dark. Different from the segmented tabs — filter tabs can be added/removed freely.
03 · NEW
Toolbar with view toggle
Group-by / assignee / due / search on the left, list/board/calendar view toggle on the right. Reusable header for any list-heavy page.
04
Reuses Admin's task group + row
Same 6-col row (checkbox / title+meta / assignee / due / [priority+status] / more) and same group card shell. Overdue dates flip to danger red text.
05
Show N more pattern
Long groups collapse after 6 rows with a "Show N more" button at the bottom. Prevents the page from stretching to 60 rows on a first load.
06
Mobile: stacked task cards
Desktop 6-col row becomes a 2-part vertical card: title + checkbox at top, "assignee · due date (X d late)" line below, priority + status pills at the bottom.
