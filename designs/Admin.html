Reflow & new components

Desktop → Mobile
Introduces the donut chart, task-group card, task row, priority badge, and status pill. All added to the shared system.
01 · NEW
Donut chart
CSS-drawn ring with segments as separate strokes and center value. Reusable for any 2–5 category distribution — legend sits to the right with swatch + name + count + %.
02 · NEW
Task group card
Card with a tinted header (title + count on the left, meta chips on the right), then a stack of task rows separated by hairlines. Reusable for any grouped list.
03 · NEW
Task row
6-column grid: checkbox, title+assignee, assignee-short, due, [priority + status], more. Big enough click target on the title; checkbox toggles complete state.
04 · NEW
Priority badge + status pill
Priority is a tight 10.5px chip (Urgent/High=danger, Medium=warning, Normal/Low=neutral). Status pill is a slightly larger pill with a dot and matching semantic color.
05
Desktop → Mobile task row
6-col row becomes a 2-part vertical card: title + meta + checkbox at top, priority + status pills below. Same information, less horizontal pressure.
06
Donut simplification
Desktop donut is 160px with a 3-item legend. Mobile drops to 96px, and the legend truncates to 2 lines with the smaller category shown as ink-500 (de-emphasized when 0).
