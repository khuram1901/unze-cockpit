Reflow & new components

Desktop → Mobile
01 · NEW
Personal minutes card
Compact meeting card with header (date + title + tags + stats) + summary block + attendees + 3-column DRO + action-item list. Always expanded for the most recent — collapsed variant for older.
02 · NEW
Section label (accent rule)
Small colored uppercase label with a 3px left rule matching semantic color (green = decisions, red = risks, blue = opportunities). Lighter alternative to the DRO cards on page 16.
03 · NEW
Bullet list
Dot-prefixed list with 12.5px text and tight line-height. Reusable for any structured content — meeting notes, requirements, checklist.
04 · NEW
Action item with "Open →" link
4-col row: checkbox, title/meta, status pill, "Open →" text link that jumps to the full task detail. Small addition to the base action item.
05
Latest expanded, rest collapsed
Only the most recent minutes are shown expanded on desktop — the rest are compact rows with title + tags + stats + "Open" button. Reduces scroll length while keeping context.
06
Mobile: card summary only
Mobile shows every meeting as a compact card with the summary paragraph inline. Tapping opens the full detail view with DRO and action items.
