Reflow & new components

Desktop → Mobile
01 · NEW
Stock hero card
Full-width dark card at the top: total in stock (large number) + inline size breakdown chips. Reused for any per-plant/per-entity aggregate.
02 · NEW
Size chip
Pill with mono unit label + display-font number. Used in the hero and in every PO card's "By size in stock" row.
03 · NEW
Plant switcher (pill tabs)
Dark active pill with count badge. Same styling as the segmented tabs from page 06 but with an inline count. Mobile becomes a scroll strip.
04
PO card (reused from Opening Balances)
Same accent-left-rule card. Green rule = fully produced; amber = partial or unallocated buffer. Meta row shows Order · Produced · Dispatched · In stock with the same 4-col grid.
05
Alert (reused)
Same warm-amber alert box used across pages 01, 06, 08, 13. Now flags authority-letter expiries.
06
Density on mobile
Desktop shows all 4 meta cells + all size chips + actions. Mobile trims to 2 meta cells (In stock, Dispatched) + size chips only. Actions move to a per-card ⋯ menu.
